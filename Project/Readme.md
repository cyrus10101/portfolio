# Introduction to Programming – Projects

This folder contains two projects completed for the Introduction to Programming course.

---

## Task 1 – Pizza Calculator

**Overview:**  
Computes the total cost of a pizza order at Beckett Pizza Plaza, applying a "Buy Four, Pay for Three" offer.

**Rules:**  
- Accept prices for four pizzas.  
- The lowest-priced pizza is free.  
- Calculates the total price and discount.  
- Validates user inputs to prevent errors.

---

## Task 2 – Password Checker

**Overview:**  
Checks the strength and validity of a user password.

**Rules:**  
- Password must have at least 9 characters.  
- Requests three random characters from the password for verification.  
- Terminates immediately if conditions are not satisfied.
