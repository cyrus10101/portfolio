# Fundamentals of Computer Programming – Coursework Portfolio

This repository contains all the coursework completed for the Computer Programming course.  
It showcases my learning journey in Python, including theoretical concepts, weekly exercises, and final projects.

---

## Repository Structure

The portfolio is organized into three main sections:

### 1. Weekly Exercise Documents
- Word documents covering programming theory and logic-based exercises.  
- Includes content for Weeks 1–8 (Week 3 is excluded).

---

### 2. Weekly Python Programs
- Python scripts created to practice programming concepts and build problem-solving skills.  
- Also contains weekly lab worksheets.

---

### 3. Final Assignment Projects

#### Beckett Pizza Plaza
A program that calculates pizza orders using a “4-for-3” promotion.

- **Features:** Validates inputs, finds the cheapest pizza, and computes the total cost.  
- **Concepts:** Lists, numeric validation, and arithmetic operations.

#### Password Security Check
A program to verify passwords by randomly checking selected characters.

- **Features:** Password length verification and character checks.  
- **Concepts:** String handling, random module, and control flow.
